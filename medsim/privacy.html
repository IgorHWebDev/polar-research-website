Privacy Policy for MedSim Bot

1. Data Collection and Usage

  • We collect user interaction data solely for educational progress tracking
  • Practice session responses are stored to provide performance analytics
  • No personal medical information is ever collected or stored

2. Data Storage and Security

  • All data is stored securely on encrypted servers
  • User data is not shared with third parties
  • Practice cases are fictional and for educational purposes only

3. User Rights

  • Users can view their data using the /progress command
  • Users can request data deletion by contacting support
  • Educational progress data is retained for 12 months

4. AI Technology

  • We use Google Vertex AI for case analysis and feedback
  • AI interactions are processed securely and not stored permanently
  • No personal identifiable information is used in AI processing
