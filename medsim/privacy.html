Privacy Policy for MedSim Bot

1. Data Collection and Usage

  • We collect user interaction data solely for educational progress tracking
  • Practice session responses are stored to provide performance analytics
  • No personal medical information is ever collected or stored

2. Data Storage and Security

  • All data is stored securely on encrypted servers
  • User data is not shared with third parties
  • Practice cases are fictional and for educational purposes only

3. User Rights

  • Users can view their data using the /progress command
  • Users can request data deletion by contacting support
  • Educational progress data is retained for 12 months

4. AI Technology and Data Sources

  • We use Google Vertex AI for case analysis and feedback
  • AI interactions are processed securely and not stored permanently
  • No personal identifiable information is used in AI processing

5. MIMIC-IV Database Attribution

  • This bot uses the MIMIC-IV clinical database from Beth Israel Deaconess Medical Center
  • All clinical cases are based on deidentified data approved for research use
  • When using this bot, please cite: Johnson, A., Bulgarelli, L., Pollard, T., Horng, S., Celi, L. A., & Mark, R. (2023). MIMIC-IV Clinical Database. PhysioNet.
  • This project was approved by the Institutional Review Boards of Beth Israel Deaconess Medical Center (Boston, MA) and the Massachusetts Institute of Technology (Cambridge, MA)

6. Disclaimer

  • This bot is for educational purposes only and should not be used for actual medical diagnosis or treatment
  • The clinical scenarios are based on deidentified historical data and should not be considered current medical advice
  • Users should always rely on their professional medical judgment and current clinical guidelines in real-world situations
